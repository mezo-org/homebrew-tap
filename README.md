# mezo-cli packages

Package manager distribution for [mezo-cli](https://github.com/mezo-org/mezo-cli), the Mezo protocol CLI.

All files in this repo are auto-generated by the mezo-cli release pipeline. Do not edit manually.

## Install

### Homebrew (macOS / Linux)

```sh
brew install mezo-org/tap/mezo-cli
```

### Scoop (Windows)

```sh
scoop bucket add mezo https://github.com/mezo-org/homebrew-tap
scoop install mezo-cli
```

### Chocolatey (Windows)

```sh
choco install mezo-cli --source https://github.com/mezo-org/homebrew-tap/tree/main/chocolatey
```

### AUR (Arch Linux)

```sh
yay -S mezo-cli-bin
```

### APT (Debian / Ubuntu)

```sh
curl -fsSL https://mezo-org.github.io/homebrew-tap/apt/key.gpg \
  | sudo gpg --dearmor -o /etc/apt/keyrings/mezo.gpg
echo "deb [signed-by=/etc/apt/keyrings/mezo.gpg] https://mezo-org.github.io/homebrew-tap/apt stable main" \
  | sudo tee /etc/apt/sources.list.d/mezo.list
sudo apt update && sudo apt install mezo-cli
```

Download binaries from [GitHub Releases](https://github.com/mezo-org/mezo-cli/releases).
